## Construction-Constraint Theory Bridge (20:28 UTC)

### New Paper: "Constraint Theory Meets Construction" (a2854c2)
Casey's insights bridging construction techniques with formal constraint theory. 

Core new ideas:
- **Lazy constraints:** "Measure the opening, cut the piece" — constraint measured at assembly not design time. Craftspeople never specify upfront — they measure what IS, then cut to fit. 
- **String measurement as analog computation:** The string IS the solver. Hull string determines frame positions without equations. String wraps 3-4-5 triangle jig to square frames. The string adapts to physical reality.
- **Pythagorean snapping in N dimensions:** Project high-dim point onto triangle basis in lower dim, snap (triangle constraints are always satisfiable), solve, project back. Dimensionality reduction AS constraint satisfaction. O(N) instead of O(N²).
- **Batten fairing:** C² continuity (position + tangent) enforced by wood's flexibility, not math. The material IS the continuity constraint. Spline with material-enforced continuity.
- **Bi/tri-fold equations:** Origami fold lines = equality constraints between reflected points. Margishvili's theorem: paper folding solves cubic equations. Peaucellier-Lipkin linkage for straight-line motion.
- **Force-fit/spring-back:** Temporal constraint — satisfy now, re-verify later. Cut slightly oversize to account for spring-back. Deferred constraint propagation.
- **PLATO screen-by-screen:** CAD snapping — frame constrains panel, panel constrains bezel. Local constraints, global coherence. String analogy: tiles as strings run along hull, local marks, global shape emerges.

### Implications for FLUX-C/PLATO:
