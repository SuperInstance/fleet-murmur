- `POST /room/{room}/submit` — Submit a Q&A tile. Body: `{ domain, question, answer, confidence }`. Returns `{ status, room, tile_hash }`.

- `DELETE /room/{room}/tile/{index}` — Delete a tile.
- Tile schema: Q&A pair with provenance chain. NOT the tile enum model.
- Requires: `domain`, `question`, `answer` fields. Min 20 chars answer, max 5000.
